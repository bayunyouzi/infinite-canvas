# 2026-06-23

## 修复 sub2api 兼容性问题

**问题**：项目文字对话使用 OpenAI Responses API（`/responses` 端点），sub2api 等中转服务不支持此端点，导致部署后无法使用自定义 API。

**修改文件**：`web/src/services/api/image.ts`

**改动内容**：
1. 新增 Chat Completions 相关类型：`ChatMessage`、`ChatToolDefinition`、`ChatCompletionsPayload`、`ChatStreamState`
2. 新增 `tochatMessages()`：将 ResponseInputMessage 格式转换为标准 messages 格式
3. 新增 `toChatTool()` / `toChatToolChoice()`：适配 tools 格式
4. 新增 `consumeChatStreamBlock()` / `consumeChatStreamText()`：解析 Chat Completions SSE 流
5. 新增 `requestChatCompletions()`：完整的 `/chat/completions` 流式请求实现（含 tool_calls 拼接）
6. 更新 `requestImageQuestion()`：改用 `requestChatCompletions`
7. 更新 `requestToolResponse()`：改用 `requestChatCompletions`

旧的 `requestStreamingResponse`（`/responses`）保留，图片生成接口不受影响。
